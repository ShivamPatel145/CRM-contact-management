# CRM — Contact Management System

> A production-grade full-stack CRM built with the MERN stack.
> Full documentation will be added in **Milestone 8**.

## Quick Start

```bash
# Backend
cd backend && cp .env.example .env  # fill in your values
npm install && npm run dev

# Frontend (new terminal)
cd frontend && npm install && npm run dev
```
